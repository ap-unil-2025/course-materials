üìù Markdown Template
  1. Download the template and save it as project_report.md
  2. Edit the file in any text editor (VS Code, Sublime, etc.)
  3. Convert to PDF using pandoc: 
    pandoc project_report.md -o project_report.pdf --pdf-engine=xelatex
  4. Alternative: Use online converters like Pandoc Online or Dillinger

üí° Tip: VS Code with Markdown Preview Enhanced extension provides live preview while editing.

üìÑ LaTeX Template
  1. Download the template and save it as project_report.tex
  2. Edit using a LaTeX editor (Overleaf, TeXstudio, VS Code with LaTeX Workshop)
  3. Compile to PDF: 
    pdflatex project_report.tex
bibtex project_report  # if using references
pdflatex project_report.tex
  4. Online option: Upload to Overleaf for cloud-based editing

üí° Tip: Overleaf is recommended for beginners - no installation required!
